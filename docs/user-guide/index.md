# 使用指南

这一栏面向两类人：想尽快把项目跑起来的人，以及只想顺着界面完成一次上课流程的人。

## 适合谁先看

- 想在 Windows 上第一次体验课狐 ClassFox 的用户
- 需要确认环境依赖与启动顺序的开发者
- 想知道“上传资料 -> 开始监听 -> 告警/救场 -> 课后总结”整条链路怎么走的人

## 建议阅读顺序

1. 先看环境准备，确认 Python、Node.js、Rust、Tauri 依赖。
2. 再看首次使用，快速完成一次监控启动。
3. 然后看功能工作流，把资料上传、救场、进度追踪和总结串起来。
4. 最后查常见问题，处理权限、音频、模型配置和运行异常。

## 这一栏包含什么

- 环境准备：本机依赖与初始化
- 首次使用：第一次跑通产品的最短路径
- 功能工作流：按真实课堂场景拆开的操作说明
- 常见问题：优先排查常见的启动与识别问题
