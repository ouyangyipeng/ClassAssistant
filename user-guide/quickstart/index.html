    首次使用¶

    适合谁¶

    如果你只是想下载成品直接使用，而不准备参与开发，这一页就足够覆盖大部分需求。

    第一次启动前¶

    1. 解压发布包到一个普通目录，不要直接在压缩包里双击运行。
    2. 确保系统麦克风可用。
    3. 如果你要使用在线模型，准备好自己的 API Key。

    启动方式¶

    双击 release 根目录的课狐ClassFox.exe。

    应用启动时会自动尝试拉起内置后端。正常情况下，你不需要再手动打开 backend 目录中的任何文件。

    建议的首次配置¶

    最小可用配置¶

    如果只是熟悉界面，可以先把 ASR_MODE 设为 mock。

    正常使用配置¶

    更常见的配置是：

    • ASR_MODE=local 或在线 ASR 模式
    • 填写 LLM_BASE_URL、LLM_API_KEY、LLM_MODEL

    这些配置可以在应用的“设置”面板中完成。

    一次标准使用流程¶

    1. 可先上传一份 PPT、PDF 或 Word 资料。
    2. 点击“开始摸鱼”。
    3. 填写课程名称，并可选择一份引用资料。
    4. 开启监听后，小窗会进入监控状态。
    5. 出现提醒时，按需选择“救场”或“老师讲到哪了”。
    6. 下课后点击结束，系统会尝试自动生成总结。

    三个核心功能怎么理解¶

    点名提醒¶

    当课堂文本命中关键词时，前端会弹出提醒层。红色通常代表高优先级，例如点名、回答、签到；黄色代表一般提醒，例如作业、重点、考试等。

    救场¶

    它会结合最近课堂内容和当前资料，推测老师刚才的问题，并给出一版可快速参考的回答。

    老师讲到哪了¶

    它不会逐句还原课堂，而是把最近一段内容压缩成面向当前状态的摘要，适合你快速跟上进度。

    使用建议¶

    • 刚开始先用 mock 或 local 模式熟悉流程。
    • 上传资料后再开始监听，救场和总结质量通常更高。
    • 结束课程后查看 data/summaries 中的总结文件，长期保留会更方便复盘。
